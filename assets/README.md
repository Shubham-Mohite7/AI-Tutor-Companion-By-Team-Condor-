# AITutor Assets

This folder contains static assets for the AITutor application.

## Contents

- **images/**: Application screenshots, logos, icons
- **icons/**: Favicon, app icons, social media images
- **banners/**: Marketing banners and promotional materials

## Usage

Assets are referenced in the frontend code using relative paths:
```typescript
import logo from '@/assets/icons/logo.png'
```

## Guidelines

- Use SVG format for icons when possible
- Optimize images for web (WebP format preferred)
- Maintain consistent branding and color scheme
- Follow accessibility guidelines for images
